# Security Policy

## Supported versions

SiteGuard is a small, fast-moving project.

Security fixes are generally applied to the latest code on the default branch and the most recent release, if releases are published.

## Reporting a vulnerability in SiteGuard

If you believe you found a vulnerability in **SiteGuard itself**:

- do **not** open a public issue with full exploit details
- prefer GitHub private vulnerability reporting or a GitHub Security Advisory draft if the repository has it enabled
- if private reporting is not available, contact the maintainer privately first and share only the minimum information needed to reproduce the problem safely

Please include as much of the following as possible:

- affected version or commit
- reproduction steps
- expected impact
- proof of concept, if safe to share
- logs, stack traces, or screenshots, if relevant

## What not to report here

This repository is for vulnerabilities in SiteGuard itself.

If SiteGuard reports a weakness on a third-party website, please do **not** report that finding to this repository unless the problem is caused by SiteGuard's scanner logic.

## Disclosure expectations

Please allow reasonable time for triage, reproduction, and a fix before public disclosure.
